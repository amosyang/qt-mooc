User Guide

Setting up the working environment

Linux

  • Install development tools (g++, make, etc.) for your distribution.
    • Ubuntu/Debian: sudo apt install build-essential
  • To run graphical applications you need to install the development libraries for OpenGL.
    • Ubuntu/Debian: sudo apt install mesa-common-dev libgl1-mesa-dev
  • You need to download and install Qt 5.12 and Qt Creator 4.8.0 with the online installer from qt.io
  • Download qtcreator-tmc-linux-v2.0.zip and extract it.
  • Copy/move the .so file extracted from the qtcreator-tmc-linux-v2.0.zip to the following path: $QT_INSTALL_DIRECTORY/Tools/QtCreator/lib/qtcreator/plugins/
  • Start Qt Creator, TestMyCode plugin should be visible on the top menu.

Windows

Windows support coming.

MacOS

  • First you should have XCode installed from the App store to provide the default MacOS compiling environment.
  • You need to download and install Qt 5.12 and Qt Creator 4.8.0 with the online installer from qt.io
  • Download qtcreator-tmc-macos-v2.0.zip and extract it.
  • Go to the folder where you installed Qt and Qt Creator with the installer.
    • Right click on Qt Creator and select "Show Package Contents".
    • Navigate to Contents/PlugIns/ under the opened package and copy libTestMyCode.dylib to this folder.
  • Start Qt Creator from the icon, TestMyCode plugin should be visible on the top menu.

Getting Started

All the functionality related to the TMC plugin for Qt Creator can be found in the dropdown menu named TestMyCode.

Start by logging in. After that choose the organization Helsingin Yliopisto and course UI Application Development with Qt and QML, new year edition, and set the Project Directory to a desirable location (this is where the exercises are going to be downloaded).

Choose which exercises to download from Download/Update. You can also launch the exercises from here. Remember that you need to configure the project in the Projects tab (left side) on the Qt Creator by choosing a kit to use.

Now it's just time to code. Once you're done you can test the project locally with Test project, and then Submit it to the TMC server for evaluation.
